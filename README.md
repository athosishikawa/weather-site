# Weather App

Este projeto é uma aplicação de clima desenvolvida com React. Ele permite que os usuários pesquisem o clima de uma cidade, visualizem as informações climáticas em tempo real, alterem unidades de temperatura (Celsius/Fahrenheit) e exibam animações com base nas condições climáticas.

## Funcionalidades

- **Pesquisa por Localização**: O usuário pode pesquisar o clima de uma cidade inserindo o nome no campo de busca.
- **Exibição de Dados Climáticos**: Mostra a temperatura, umidade, velocidade do vento, sensação térmica e a descrição do clima.
- **Alteração de Unidades**: O usuário pode alternar entre Celsius e Fahrenheit para ver a temperatura nas unidades desejadas.
- **Animações Dinâmicas**: O clima é representado com animações visuais usando `Lottie` com base nas condições meteorológicas.
- **Mensagens de Erro**: Em caso de erro na requisição, o aplicativo exibirá uma mensagem de erro, informando ao usuário sobre a falha.

## Tecnologias Utilizadas

- **React**: Para construção da interface de usuário.
- **Axios**: Para realizar requisições HTTP à API de clima.
- **Framer Motion**: Para animações e transições fluidas na interface.
- **Lottie**: Para animações baseadas nas condições climáticas.
- **OpenWeather API**: Para obter dados climáticos em tempo real.
